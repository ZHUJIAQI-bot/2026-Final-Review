# 2026 春学期期末考试复习资料

这个仓库用于公开分享期末复习整理内容，目前包含：

- 操作系统
- 数据库原理
- 计算机组成
- 计算机网络
每个学科目录中包含：

- 复习思维导图 Markdown
- Obsidian 白板 Canvas
- 模拟试卷
- 资料勘误与考点分析
- 往年回忆版真题
  
## 使用方式

1. 直接在 GitHub 网页查看 Markdown 文件。
2. 用 Obsidian 打开本仓库目录，可以查看 `.canvas` 白板。
3. 若要同步更新，推荐用 Obsidian Git 插件或普通 Git 命令。
