# 计算机组成考点分析

> 整理依据：第 1-10 章上课 PPT、2024-2025 学年计算机组成回忆版试卷、你发的作业题截图。老师没有单独划重点，因此优先级按“去年卷面出现频率 + 作业题计算密度 + PPT章节核心程度”综合判断。

## 一、去年回忆卷题型画像

去年回忆版为 120 分钟，题型全部是解答题，共 11 个大题。题目不是简单背诵，主要考：画图、字段划分、公式计算、过程说明、控制信号/微操作补全。

高频能力要求：

1. 能画结构图或时空图：存储器连接逻辑图、流水线时空图、中断运行轨迹、DMA地址布局。
2. 能按位划分地址字段：主存地址、Cache地址、片选地址线、块内地址、组号、标记位。
3. 能写硬件过程：中断响应、DMA预处理/传送/后处理、取指/执行周期微操作。
4. 能做典型计算：Cache命中率与平均访问时间、IEEE754、流水线性能、磁盘容量/位密度/传输率。
5. 能比较结构优缺点：单总线瓶颈、程序查询/中断/DMA、RISC/CISC、组合逻辑/微程序控制。

## 二、A 级必练考点

### 1. 存储器扩展与片选译码

去年卷直接考：用 1K x 4 bit SRAM 组成 5K x 16 bit 存储模块。

必须会写：

- 位扩展：4 片 1K x 4 并联形成 1K x 16。
- 字扩展：5 组 1K x 16 形成 5K x 16。
- 地址线：芯片内 1K 需要 10 根地址线；若系统按字节编址且 16 位数据总线，一般 A1-A10 接芯片内部地址，A0 表示字节位置或要求字对齐。
- 片选线：用更高位地址线译码产生 CS0-CS4。
- 地址范围：若模块从 0000H 开始，5 组每组 2KB，范围为 0000H-27FFH。

易错点：字编址和字节编址不要混。去年题明确“按字节编址，16位数据总线，每次访问一个字=2字节”，因此地址范围按字节算。

### 2. Cache 地址格式、命中率与效率

去年卷直接考：8MB 主存、32KB Cache、32B 块，全相联与四路组相联字段划分，并求地址映射组号。

作业截图也多次出现 Cache：命中率、平均访问时间、效率、组相联字段划分。

必背公式：

- 块内地址位数：b = log2(块大小)
- Cache 行数：C = Cache容量 / 块大小
- 组数：Q = Cache行数 / 路数
- 组号位数：q = log2(Q)
- 标记位：tag = 主存地址位数 - 组号位数 - 块内地址位数
- 命中率：h = 命中次数 / 总访问次数
- 平均访问时间按课件公式：ta = h tc + (1-h) tm
- Cache效率：e = tc / ta x 100%

注意：有些教材把未命中时间写成 tc + tm，此时 ta = htc + (1-h)(tc+tm)。考试以老师 PPT 公式为准，作业截图采用的是 htc + (1-h)tm。

### 3. 中断系统与多重中断

去年卷两次考中断：

- CPU 何时检查中断请求，何时响应中断。
- 用中断屏蔽字改变实际处理优先级，并画 CPU 运行轨迹。

必须会写：

- CPU 通常在一条指令执行结束时检查中断请求，保证当前指令完整执行，便于保存断点和恢复现场。
- 响应条件：有中断请求；中断允许触发器为 1；该级中断未被屏蔽；优先级判优通过；通常还要处于可响应中断的时刻。
- 响应过程：关中断、保存断点和现场、判优、形成中断服务程序入口、执行服务程序、恢复现场、开中断并返回。
- 多重中断：在中断服务程序中重新开中断，靠屏蔽字决定哪些更高处理优先级中断可以嵌套。

### 4. DMA 方式

去年卷直接考：DMA 控制器寄存器配置、地址自动递增、完成判断、通知 CPU、主存布局图。

必须会写：

- AR：主存起始地址。
- WC：传送字数或字节数，取决于一次传送单位。
- DAR：设备或磁盘控制器 I/O 地址。
- 控制寄存器：传送方向、传送方式、启动位、中断允许等。
- 过程：预处理 -> 数据传送 -> 后处理。
- 完成判断：字计数器减到 0 或溢出/终止信号；完成后向 CPU 发中断请求。

### 5. IEEE754 单精度浮点数

去年卷直接考：把 3.75 转为 32 位 float 并写十六进制。

步骤模板：

1. 写成二进制并规格化：3.75 = 11.11B = 1.111B x 2^1。
2. 符号位 S = 0。
3. 阶码 E = 1 + 127 = 128 = 10000000B。
4. 尾数字段 M = 111000...0，共 23 位。
5. 合并为 0 10000000 11100000000000000000000 = 40700000H。

### 6. 指令寻址方式与有效地址

去年卷直接考：立即、间接、相对、变址四种寻址方式求 E。

模板：

- 立即寻址：操作数在指令地址字段 D 中，一般不再计算有效地址。
- 直接寻址：E = D。
- 间接寻址：E = (D)。
- 相对寻址：E = (PC) + D。三字节指令从 B 开始时，取完指令后 PC = B + 3，所以 E = B + 3 + D。
- 变址寻址：E = D + Z。

### 7. 指令流水线

去年卷考两类：

- 给出指令冲突例子，说明涉及寄存器和后果。
- 给四段流水线时间，画前 4 条指令时空图，算总时间、吞吐率、加速比。

公式模板：

- 流水线节拍：Delta t = max(各段执行时间 + 段间寄存器延迟)
- k 段流水线执行 n 条指令总时间：T = (k+n-1) Delta t
- 吞吐率：TP = n / T
- 非流水顺序时间：T0 = n x sum(各段时间)
- 加速比：S = T0 / T

### 8. 控制单元微操作

去年卷考存数指令执行周期微操作和控制信号补全。

必须掌握：

- 取指周期：PC -> MAR；1 -> R；M(MAR) -> MDR；MDR -> IR；OP(IR) -> CU；(PC)+1 -> PC。
- 存数 STA X 执行周期：Ad(IR) -> MAR；1 -> W；AC -> MDR；MDR -> M(MAR)。
- 加法 ADD X 执行周期：Ad(IR) -> MAR；1 -> R；M(MAR) -> MDR；(AC)+(MDR) -> AC。
- 间址周期：Ad(IR) -> MAR；1 -> R；M(MAR) -> MDR；MDR -> Ad(IR)。

## 三、B 级常考考点

1. 单总线结构瓶颈：共享总线竞争、CPU/主存/I/O不能充分并行、带宽不足、等待时间增加、扩展设备后冲突严重、系统可靠性和实时性下降。
2. 系统总线：数据总线、地址总线、控制总线；同步、异步、半同步、分离式通信；总线判优与带宽。
3. 程序查询、中断、DMA比较：CPU参与程度、传送单位、并行性、速度、硬件复杂度、适用对象。
4. RISC/CISC：指令复杂度、长度、寻址方式、访存限制、控制器实现、流水线适应性。
5. 组合逻辑控制与微程序控制：速度、灵活性、修改难度、设计复杂度。

## 四、复习顺序

1. 先刷计算题：Cache、存储器扩展、IEEE754、流水线、磁盘。
2. 再背过程题：中断响应、DMA流程、指令周期、微操作。
3. 最后整理比较题：单总线、I/O三方式、RISC/CISC、控制单元设计。

## 五、答题格式建议

- 计算题：公式 -> 代入 -> 结果 -> 单位 -> 一句话解释。
- 画图题：先标字段和线名，再写每根线的作用。
- 过程题：按时间顺序写，不要只写关键词。
- 比较题：用表格从 4-6 个维度比较，容易拿满。
