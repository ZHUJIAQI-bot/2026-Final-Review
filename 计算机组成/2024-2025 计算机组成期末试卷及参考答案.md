# 参考答案

说明：第 11 题原 PDF 的 CPU 图和控制信号编号无法从文本中完整提取，答案按常见存数指令微操作整理，控制信号编号需结合课堂图确认。

## 1、流水线冲突

例：

```asm
I1: ADD R1, R2, R3
I2: SUB R4, R1, R5
```

I2 要读 R1，而 I1 尚未把结果写回 R1，存在 RAW 写后读数据相关，涉及寄存器 R1。若不解决，I2 会读到旧值，导致结果错误。可用暂停、插入气泡、数据前递、编译器调度等方法解决。

## 2、中断检查与响应

CPU 通常在一条指令执行结束后、下一条指令取指前检查中断请求。这样可保证指令完整执行，并让 CPU 在一致状态下保存断点和现场。

响应条件：有中断请求；CPU 允许中断；该中断源未被屏蔽；优先级满足响应条件；当前指令已结束。

响应过程：检查请求和屏蔽字；保存 PC 和 PSW；关中断或改变中断允许状态；根据中断向量转入服务程序；服务结束后恢复现场并返回。

## 3、单总线结构问题

多 I/O 设备频繁交互时，单总线会出现总线竞争严重、CPU 访存和 I/O/DMA 传输互相阻塞、慢速设备拖累整体效率、总线负载大、仲裁复杂、扩展性差、总线故障影响范围大等问题。

## 4、SRAM 存储器扩展

用 `1K×4bit` 组成 `5K×16bit`：

```text
位扩展：4bit -> 16bit，需要 4 片并联组成 1 组
字扩展：1K -> 5K，需要 5 组
总芯片数 = 20 片
```

每组 4 片分别连接 D0-D3、D4-D7、D8-D11、D12-D15。每片 SRAM 片内地址为 10 位。按字节编址且 16 位数据总线时，A0 作字节选择；A1-A10 作片内寻址；A11 及以上作组片选译码。

每组容量 `1K×16bit = 2KB = 0800H`。若从 0000H 开始：

| 组号 | 地址范围 |
|---:|---|
| 0 | 0000H-07FFH |
| 1 | 0800H-0FFFH |
| 2 | 1000H-17FFH |
| 3 | 1800H-1FFFH |
| 4 | 2000H-27FFH |

`0A3CH` 位于 `0800H-0FFFH`，属于 1 组。

升级为 `8K×16bit`：需要 8 组、32 片；片内寻址仍为 A1-A10；组选择改为 A11-A13 接 3-8 译码器，输出 8 个片选信号。

## 5、Cache 地址划分

```text
主存 8MB = 2^23B，地址 23 位
Cache 32KB = 2^15B
块大小 32B = 2^5B，块内地址 5 位
Cache 行数 = 2^15 / 2^5 = 2^10 = 1024
```

全相联：

```text
Tag 18 位 | 块内地址 5 位
```

四路组相联：

```text
组数 = 1024 / 4 = 256 = 2^8
Tag 10 位 | 组号 8 位 | 块内地址 5 位
```

地址 `1B4F3CH`：

```text
组号 = (1B4F3CH >> 5) & FFH = 79H = 121
```

映射到第 0x79 组，即十进制 121 组。

## 6、中断屏蔽字与运行轨迹

实际处理优先级为 `1 > 4 > 3 > 2`，屏蔽位 0 开放、1 屏蔽。原则是开放更高级，屏蔽自身及更低级。

| 中断服务程序 | 屏蔽字 |
|---|---|
| 1 级 | 1111 |
| 4 级 | 0111 |
| 3 级 | 0110 |
| 2 级 | 0100 |

运行轨迹：

```text
主程序 -> 3级ISR -> 4级ISR -> 1级ISR -> 4级ISR续 -> 3级ISR续 -> 2级ISR -> 主程序
```

解释：3、4 同时请求时按硬件优先级先响应 3；3 级服务中开放 4，所以 4 可打断 3；4 级中只开放 1，所以 1 打断 4，2 等待。

## 7、DMA

总传输量 `4*512=2048B`，16 位总线一次传 2B：

```text
AR = 20000H
WC = 2048 / 2 = 1024 = 0400H
DAR = 磁盘控制器数据端口地址，题目未给具体数值
控制寄存器 = 磁盘 -> 主存，读磁盘/写主存，设置 DMA 传送模式
```

每传 1 个字：写入当前 AR，随后 `AR=AR+2`，`WC=WC-1`。当 WC=0 或终端计数信号有效时，DMA 结束，释放总线并向 CPU 发中断。

写入区域：

```text
20000H - 207FFH
```

## 9、寻址方式

三字节指令起始地址 B，地址字段 D，变址寄存器 Z。

| 寻址方式 | E |
|---|---|
| 立即寻址 | 操作数就是 D，通常不计算有效地址 |
| 间接寻址 | E = (D) |
| 相对寻址 | E = B + 3 + D |
| 变址寻址 | E = D + Z |

## 10、流水线计算

流水线节拍：

```text
Δt = max(3,4,6,3) = 6ns
```

时空图：

```text
        1拍   2拍   3拍   4拍   5拍   6拍   7拍
I1      S1    S2    S3    S4
I2            S1    S2    S3    S4
I3                  S1    S2    S3    S4
I4                        S1    S2    S3    S4
```

```text
T流水 = (4 + 60 - 1) * 6ns = 378ns
T非流水 = 60 * (3+4+6+3)ns = 960ns
TP = 60 / 378ns ≈ 0.1587 条/ns = 1.587 * 10^8 条/s
S = 960 / 378 ≈ 2.54
```

## 11、存数指令微操作

存数指令执行周期通常为：

```text
Ad(IR) -> MAR
1 -> W
AC -> MDR
MDR -> M(MAR)
```

如果表格要求控制信号编号，需结合课堂 CPU 图填写；考试关键是补出 `AC -> MDR` 和 `MDR -> M(MAR)`。
