# 计算机组成期末模拟试卷（全解答题）

闭卷/开卷均可练习 | 建议用时 120 分钟 | 满分 120 分

> 命题依据：第 1-10 章 PPT、2024-2025 回忆版试卷、作业截图高频题。题目全部按解答题设计，重点训练写过程、画图、列公式、说明理由。

## 试题

### 第一题 流水线冲突与性能计算（12分）

某处理器采用四级流水线 S1、S2、S3、S4，各段延迟分别为 3ns、4ns、6ns、3ns，段间寄存器延迟忽略。连续执行 60 条指令。

1. 确定流水线节拍 Delta t，并画出前 4 条指令的时空图。
2. 计算总执行时间、实际吞吐率 TP、相对于非流水顺序执行的加速比 S。
3. 举一个会产生数据相关冲突的指令序列，说明涉及哪些寄存器，以及不处理会造成什么错误。

### 第二题 中断响应与屏蔽字（12分）

某系统有 4 个中断源，硬件响应优先级为 1 > 2 > 3 > 4。现要求实际处理优先级变为 1 > 4 > 3 > 2。屏蔽位 0 表示开放，1 表示屏蔽。

1. 说明 CPU 通常在什么时候检查中断请求，为什么这样设计。
2. 说明 CPU 响应中断请求需要满足哪些条件，并写出响应过程。
3. 为 1、2、3、4 级中断服务程序分别设计屏蔽字。
4. 若主程序运行时 3、4 同时请求；执行 4 级服务程序时 1、2 先后请求，说明 CPU 运行轨迹。

### 第三题 单总线结构分析（8分）

早期小型计算机常采用单总线结构，即 CPU、主存和所有 I/O 设备挂接在同一组系统总线上。请分析在多 I/O 设备与 CPU 频繁交换数据时，单总线结构会暴露哪些性能瓶颈和严重问题。

### 第四题 存储器扩展与片选译码（14分）

现使用 1K x 4 bit 的 SRAM 芯片，组成 5K x 16 bit 的存储模块。系统按字节编址，数据总线 16 位，每次访问 1 个字，即 2 字节。假定模块从物理地址 0000H 开始。

1. 需要多少片 SRAM？如何进行位扩展和字扩展？
2. 说明 CPU 地址线、数据线、片选信号如何连接。
3. 写出各组芯片在主存中的十六进制地址范围。
4. 地址 0A3CH 对应哪一组芯片？说明判断过程。
5. 若升级为 8K x 16 bit，片选译码逻辑如何修改？

### 第五题 Cache 地址划分与映射（12分）

某主存-Cache 系统按字节编址。主存容量 8MB，Cache 容量 32KB，Cache 块大小 32B。

1. 采用全相联映射时，画出主存地址字段格式并写出各字段位数。
2. 采用四路组相联映射时，画出主存地址字段格式并写出各字段位数。
3. 在四路组相联下，物理地址 1B4F3CH 映射到 Cache 的哪一组？
4. 简述写直达法和写回法的区别。

### 第六题 Cache 命中率与平均访问时间（10分）

CPU 执行程序时，Cache 命中 4800 次，访问主存 200 次。Cache 存取周期为 30ns，主存存取周期为 150ns。

1. 求 Cache 命中率。
2. 按本课程 PPT 公式求 Cache-主存系统平均访问时间和效率。
3. 与无 Cache 时相比，系统速度变为原来的多少倍？提高了多少倍？

### 第七题 DMA 控制器配置与流程（12分）

某系统用 DMA 控制器把磁盘连续 4 个扇区传送到主存。每扇区 512B，共 2048B。系统数据总线 16 位，DMA 一次传送 2B。主存写入起始地址为 20000H。

1. 配置 DMA 的 AR、WC、DAR、控制寄存器，并说明理由。
2. 说明 DMA 如何通过地址自动递增完成连续写入。
3. 说明 DMA 如何判断传送完成，如何通知 CPU。
4. 写出主存中被写入的地址范围。

### 第八题 IEEE754 单精度浮点表示（8分）

float 型变量 y = 3.75，采用 IEEE754 单精度格式，存入 32 位浮点寄存器 FR2。请写出 FR2 的二进制内容和十六进制表示，并写出计算过程。

### 第九题 寻址方式与有效地址（8分）

主存按字节编址。一条三字节指令起始地址为 B，地址字段 D 存在 B+1、B+2 处，变址寄存器内容为 Z。分别写出以下寻址方式下操作数或有效地址 E 的表达式。

1. 立即寻址。
2. 直接寻址。
3. 间接寻址。
4. 相对寻址。
5. 变址寻址。

### 第十题 磁盘性能计算（10分）

某磁盘转速 3000 转/分，共 4 个记录盘面，每毫米 5 道，每道记录信息 12288 字节，最小磁道直径 230mm，共 275 道。

1. 求磁盘存储容量。
2. 求最高位密度和最低位密度。
3. 求磁盘数据传输率。
4. 求平均旋转等待时间。

### 第十一题 控制单元微操作（14分）

某简单 CPU 有 PC、MAR、MDR、IR、AC 和 CU。存数指令 STA X 的功能是把 AC 的内容写入主存地址 X。

1. 写出取指周期微操作序列。
2. 写出 STA X 执行周期微操作序列。
3. 写出 ADD X 执行周期微操作序列。
4. 比较组合逻辑控制器和微程序控制器。

## 参考答案与评分要点

### 第一题

1. Delta t = max(3,4,6,3) = 6ns。前 4 条指令时空图可写为：

| 时间节拍 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|---|
| I1 | S1 | S2 | S3 | S4 |  |  |  |
| I2 |  | S1 | S2 | S3 | S4 |  |  |
| I3 |  |  | S1 | S2 | S3 | S4 |  |
| I4 |  |  |  | S1 | S2 | S3 | S4 |

2. k=4，n=60，T=(k+n-1)Delta t=(4+60-1)x6=378ns。TP=60/378 条/ns。非流水 T0=60x(3+4+6+3)=960ns。S=960/378=2.54。
3. 例：ADD R1,R2,R3；SUB R4,R1,R5。第二条要读 R1，第一条尚未写回 R1，产生 RAW 数据相关。不处理会让第二条读到旧 R1，结果错误。可用暂停、转发/旁路、编译器重排解决。

### 第二题

1. CPU 通常在一条指令执行结束时检查中断请求。这样可保证指令执行的原子性，便于保存断点和恢复现场。
2. 条件：有中断请求；中断允许触发器为 1；该级中断未被屏蔽；优先级判优通过；CPU处于可响应时刻。过程：关中断 -> 保存断点和现场 -> 判优 -> 形成入口地址 -> 执行服务程序 -> 恢复现场 -> 开中断 -> 返回。
3. 实际处理优先级 1 > 4 > 3 > 2 时，某级服务程序应开放比自己实际优先级高的中断，屏蔽自己及更低级中断。一种屏蔽字为：

| 服务程序 | 屏蔽字(1,2,3,4) |
|---|---|
| 1级 | 1111 |
| 4级 | 0111 |
| 3级 | 0110 |
| 2级 | 0100 |

4. 若主程序运行时 3、4 同时请求，按硬件响应优先级先响应 3；装入 3 级屏蔽字后，4 级开放并可中断 3 级。随后 1、2 在 4 级服务中请求，1 级可中断 4 级，2 级被屏蔽等待。轨迹可写为：主程序 -> 3级入口 -> 4级服务 -> 1级服务 -> 返回4级 -> 返回3级 -> 2级服务 -> 主程序。

### 第三题

单总线是共享传输介质，CPU、主存、I/O 设备不能同时占用总线。多设备频繁交换时会出现：总线竞争严重；CPU 访存和 I/O 传送互相等待；主存带宽被 I/O 占用；高速设备难以发挥速度；实时设备等待时间不可控；设备增加后总线负载加重、仲裁复杂；单总线故障影响全系统。

### 第四题

1. 1K x 4 扩为 1K x 16 需要 4 片并联；5K x 16 需要 5 组；总片数 4x5=20 片。
2. 每片 1K 需要 10 根片内地址线。系统按字节编址且 16位数据总线，通常 A1-A10 接芯片地址输入，A0 用于字节选择或要求字对齐。D15-D0 分给每组 4 片：每片接 4 位数据线。高位地址 A11-A13 送 3-8 译码器，输出 CS0-CS4 选择 5 组。
3. 每组 1K 字 = 2KB，地址范围：0组 0000H-07FFH；1组 0800H-0FFFH；2组 1000H-17FFH；3组 1800H-1FFFH；4组 2000H-27FFH。
4. 0A3CH 位于 0800H-0FFFH，属于 1 组。也可看 A13A12A11 = 001，对应 CS1。
5. 升级为 8K x 16 需要 8 组，共 32 片。片选译码器使用 A13-A11 的全部 3 位，输出 CS0-CS7；地址范围扩展到 0000H-3FFFH。

### 第五题

1. 主存 8MB = 2^23 B，块大小 32B，块内地址 5 位。全相联无行号/组号，格式：标记 18 位 + 块内地址 5 位。
2. Cache 行数 = 32KB/32B = 1024 = 2^10。四路组相联组数 = 1024/4 = 256 = 2^8。格式：标记 10 位 + 组号 8 位 + 块内地址 5 位。
3. 组号 = (1B4F3CH >> 5) & FFH = 79H = 121。
4. 写直达：写 Cache 时同时写主存，一致性好但写速度受主存限制。写回：只写 Cache，替换时再写回主存，速度快但控制复杂，需要脏位。

### 第六题

1. h=4800/(4800+200)=0.96。
2. 按 PPT：ta = htc + (1-h)tm = 0.96x30 + 0.04x150 = 34.8ns。e = tc/ta x100% = 30/34.8 x100% = 86.2%。
3. 无 Cache 访问时间按主存 150ns 计，加速比 = 150/34.8 = 4.31。速度变为原来的 4.31 倍；若按“提高了多少倍”表述，则提高约 3.31 倍。

### 第七题

1. AR=20000H。总字节数 2048B，一次传送 2B，所以 WC=2048/2=1024 字。DAR=磁盘控制器中对应起始扇区或数据端口地址。控制寄存器设置为磁盘读、I/O到主存、16位传送、启动DMA，可选周期挪用或题目指定方式。
2. 每完成一次 2B 传送，AR 自动加 2，WC 减 1；下一次把数据写入下一个字地址，直到连续区域写满。
3. WC 减到 0 或产生终止/溢出信号表示完成。DMA 结束后释放总线，向 CPU 发中断请求，由 CPU 执行后处理程序。
4. 写入地址范围为 20000H-207FFH，共 2048B。

### 第八题

3.75 = 11.11B = 1.111B x 2^1。符号位 S=0；阶码 E=1+127=128=10000000B；尾数字段为 111 后补 0 到 23 位。二进制为：0 10000000 11100000000000000000000。十六进制为 40700000H。

### 第九题

1. 立即寻址：操作数就是 D，通常不计算有效地址 E。
2. 直接寻址：E=D。
3. 间接寻址：E=(D)。
4. 相对寻址：取完三字节指令后 PC=B+3，因此 E=B+3+D。
5. 变址寻址：E=D+Z。

### 第十题

1. 容量 = 275 x 12288B x 4 = 13516800B。
2. 最高位密度在最小磁道：12288B/(230π mm)=17B/mm=136bit/mm。最大磁道直径=230+275/5x2=340mm，最低位密度=12288B/(340π mm)=11.5B/mm=92bit/mm。
3. 3000转/分=50转/秒，传输率=12288B x 50 = 614400B/s。
4. 平均旋转等待时间 = 1/(2x50)=0.01s=10ms。

### 第十一题

1. 取指周期：PC -> MAR；1 -> R；M(MAR) -> MDR；MDR -> IR；OP(IR) -> CU；(PC)+1 -> PC。
2. STA X 执行周期：Ad(IR) -> MAR；1 -> W；AC -> MDR；MDR -> M(MAR)。
3. ADD X 执行周期：Ad(IR) -> MAR；1 -> R；M(MAR) -> MDR；(AC)+(MDR) -> AC。
4. 组合逻辑控制器由门电路直接产生控制信号，速度快，但设计和修改困难，适合 RISC 等规整指令系统。微程序控制器用微指令产生控制信号，设计规整、易修改、扩展方便，但速度通常慢于组合逻辑控制。
